# Xray-Argo 精简版一键脚本

精简高效的 Xray-Argo 一键安装脚本，无交互全自动部署。

## 支持系统

Debian / Ubuntu / CentOS / Alpine / Fedora / AlmaLinux / Rocky Linux / Amazon Linux

## 使用方法

**一键安装（交互菜单）**
```bash
bash <(curl -Ls https://raw.githubusercontent.com/Luckylos/xray-2go/refs/heads/main/xray_2go.sh)
```
